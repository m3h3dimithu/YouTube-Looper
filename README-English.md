# 🔁 YouTube Auto Looper
**Custom Browser Extension for Seamless Music Experience**

When listening to random songs or music on YouTube, there are times when we want to listen to a specific track repeatedly. By default, YouTube either stops the video when it ends or moves to the next one if Autoplay is on. To solve this, **YouTube Auto Looper** was created. It adds a sleek 'Loop' button to your YouTube player, allowing you to easily loop any video.

## 🌟 Key Features
- **One-Click Toggle:** A dedicated loop button right in the player controls.
- **Visual Feedback:** The icon turns **Green** when loop is active and stays **Red** (YouTube's signature color) when inactive.
- **Perfect Alignment:** Designed to match perfectly with existing YouTube controls like CC and Settings.
- **Multi-Browser Support:** Successfully tested on Google Chrome and Brave Browser.

## 🛠️ The Story Behind the Project
I usually listen to random music on YouTube and often find a song I want to hear on repeat. Tired of manually clicking replay, I decided to build a solution. With the assistance of Gemini (AI), and after several rounds of code modification and design iterations, I successfully implemented this loop button. This project isn't just code; it's a small tool to enhance my daily music experience.

## 🚀 How to Install
1. Download or Clone this repository.
2. Open your browser's (Chrome/Brave) **Extensions** page.
3. Enable **Developer Mode** from the top right corner.
4. Click on **Load unpacked** and select the `YouTube-Looper` folder.
5. Go to YouTube and enjoy your music!
